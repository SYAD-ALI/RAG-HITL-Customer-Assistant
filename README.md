# RAG Customer Support Assistant

## Overview

This project is a **Retrieval-Augmented Generation (RAG) based Customer Support Assistant** developed as part of the Innomatics Research Labs internship project.

The system is designed to process PDF knowledge base documents, retrieve relevant information using embeddings, and generate contextual responses to user queries. It also includes **LangGraph workflow orchestration** and **Human-in-the-Loop (HITL)** escalation for complex queries.

---

## Features

* PDF document ingestion and processing
* Text chunking for efficient retrieval
* Embedding generation using Sentence Transformers
* Vector storage using ChromaDB
* Context-aware question answering
* LangGraph based workflow management
* Conditional routing logic
* Human-in-the-Loop escalation support
* Google Colab compatible implementation
